# tictactoepro-2879

TicTacToePro is an advanced, real-time multiplayer Tic-Tac-Toe platform that enhances the classic game with modern features like competitive leaderboards, customizable game modes, and seamless social interactions. It allows players to challenge friends or random opponents, track stats, and earn achievements, making it engaging for both casual and competitive users.

## Tech Stack
React (Vite) + Tailwind CSS + Supabase

## Setup
Run `docker-compose up` to start the infrastructure.
